Course Match
  • Python, React, machine learning

A prototype web application to compare course outlines between universities. I used a support vector machine with sklearn to classify sentences in course outlines. On similarly classified sentences, I ran the RAKE algorithm to extract keywords, and then used cosine distance between vectorised keywords to determine the similarity.
